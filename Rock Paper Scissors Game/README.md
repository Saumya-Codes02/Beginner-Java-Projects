# 🎮 Rock Paper Scissors Game (Java)

A simple **console-based Rock Paper Scissors** game built in **Java**. The player competes against the computer, which makes random choices each round.

---

## 📌 Features

* 🎮 Play Rock, Paper, Scissors against the computer
* 🤖 Random computer moves
* ✅ Input validation
* 🔁 Play multiple rounds
* 📊 Live score tracking
* 🏆 Final match summary
* 🧹 Clean and beginner-friendly Java code

---

## 🛠️ Technologies Used

* Java
* IntelliJ IDEA / Eclipse / VS Code
* Java Scanner
* Java Random

## 🎲 Gameplay

```
==================================
     ROCK PAPER SCISSORS
==================================

Enter your move (rock, paper, scissors): rock

You chose      : Rock
Computer chose : Paper

Computer wins!

---------- SCORE ----------
Wins   : 0
Losses : 1
Ties   : 0
---------------------------

Play again? (yes/no):
```

---

## 📖 Rules

* 🪨 Rock beats Scissors
* 📄 Paper beats Rock
* ✂️ Scissors beats Paper
* Same choices result in a tie.

---

## ✅ Input Validation

The game accepts only:

* `rock`
* `paper`
* `scissors`

For replay:

* `yes`
* `no`

Any other input displays an error message and prompts again.

---

## 📊 Score Tracking

The game records:

* Wins
* Losses
* Ties

A final summary is displayed when the player exits.
