# Java Quiz Game 🎯

A simple console-based quiz game built using Java. The application asks multiple-choice questions, validates user input, calculates the final score, and displays the user's performance.

## 📌 Features

* Multiple-choice quiz questions
* Input validation
* Instant feedback for each answer
* Displays the correct answer when the user is wrong
* Final score summary
* Percentage calculation
* Performance grading
* Beginner-friendly Java project

## 🛠️ Technologies Used

* Java
* Scanner Class
* Arrays (1D & 2D)
* Loops
* Conditional Statements

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Java-Quiz-Game.git
```

### 2. Navigate to the project folder

```bash
cd Java-Quiz-Game
```

### 3. Compile the program

```bash
javac Main.java
```

### 4. Run the program

```bash
java Main
```
